## Objetivo

O projeto tem como objetivo implementar as funcionalidades básicas de um sistema de gestão de estádios, permitindo cadastrar, listar, atualizar e remover estádios. O sistema foi desenvolvido utilizando Python com a biblioteca Streamlit para a interface do usuário e MongoDB como banco de dados.

## Estrutura do Projeto

O repositório está organizado da seguinte forma:

- **main.py**: Arquivo principal que contém a lógica da aplicação e a interface do usuário.
- **mongodb/CRUD-estadios.js**: Script de teste para o MongoDB.
- **README.md**: Este arquivo.

## Requisitos

- Python 3.6+
- Streamlit
- MongoDB

## Instalação e Execução

1. Clone o repositório:
   ```bash
   git clone https://github.com/pedroailton/iaad-equipe-1.git
   ```

2. Instale as dependências:
   ```bash
   pip install streamlit
   ```

3. Execute a aplicação:
   ```bash
   python -m streamlit run interface/app.py
   ```
